Project Title

Online Food Ordering System

Project Description

The Online Food Ordering System is a web-based system that helps manage food menu items digitally. It allows the admin to add menu items, display available foods, update food prices, and delete unavailable items using CRUD operations connected to a database.

Detailed System Flow

1. Add Menu Items
   
-Admin adds food name, category, price, and description. 

-System saves the food item in the database and displays it on the menu. 

2. Display Menu
   
-Users open the menu page. 

-System displays all available food items with prices and descriptions. 

3. Update Food Prices
   
-Admin selects a food item and edits the price or details. 

-System updates the information in the database automatically. 

4. Delete Unavailable Items
   
-Admin removes unavailable or out-of-stock food items. 

-System deletes the selected item from the menu and database.
